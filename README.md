## 需要安装的包
### 1. selenium
`python3 -m pip install selenium`
### 2. yaml配置文件读取组件
`python3 -m pip install pyyaml`
### 3. 日志框架
`python3 -m pip install loguru`
### 4. 定时任务框架
`python3 -m pip install apscheduler`
### 5. 安装图片处理器PIL
`python3 -m pip install pillow`
### 6 安装百度API
`python3 -m pip install baidu-aip`
